speaker
=======
Модуль для организации диалога с компьютером на естественном языке.

Отечает на вопросы пользователей.
Задачи: 
а) анализ вопроса;
б) синтез ответа.

Дополнительные задачи:
а) ввод вопроса голосом и распознавание речи (пока что просто в текстовое поле вопроса в Хроме добавлен микрофон);
б) ответ голосом (синтез речи).
